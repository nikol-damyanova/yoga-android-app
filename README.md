# True Pose - Yoga App for Android
- This app allows you to select the length for your yoga session and shuffles through multiple yoga poses using a randomizer

  Features:
  - Completely Ad-free
  - Use without an account
  - Personalized work-out sessions
  - Option to pouse, skip or return to as asana
